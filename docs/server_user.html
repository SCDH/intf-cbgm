Server¶

server/server.py¶

API server for CBGM.

usage: server/server.py [-h] [-v] [-c CONFIG_PATH]
-h, --help¶

show this help message and exit

-v, --verbose¶

increase output verbosity

-c <config_path>, --config-path <config_path>¶

the directory where the config files reside (default=’./instance/’)
